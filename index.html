Tipos de datos en Javascript

  • 	  const primer_bool = true 
	  typeof primer_bool  // boolean
	  const segundo_bool = false 
	  typeof segundo_bool  // boolean
    Boolean

    Boolean es un tipo de dato logico con el que puedes solo los valores true y false. 

  • 
	  const una_variable = null 
	  typeof una_variable  // null
    null

    Un valor null representa a la referencia a la que apunta. 

  • 
	  const variable_indefinida = undefined 
	  typeof variable_indefinida  // undefined
    undefined

    Valor que se le auto asigna a las variables acaban de ser declaradas. 

  • 
	  const un_numero = 420 
	  typeof un_numero  // number
    Number

    Este tipo de dato es un numero decimal con precision de 64 bits. 

  • 
	  const numero_gigante = 31415629565n 
	  typeof numero_gigante  // bigint
    Bigint

    Este tipo de dato puede representar numeros en el formato de precision arbritaria. 

  • 
	  const string = 'Hola mundo' 
	  typeof string  // string
    String

    Un String es una secuencia de caracteres usada para representar un texto. 

  • 
	  const symbol = new Symbol() 
	  typeof symbol  // symbol
    Symbol

    Este es un build-in cuyo constructor devuelve un simbolo primitivo que se garantiza ser unico. 

  • 	  const un_objeto = new Object()
	  const otro_objeto = {} 
	  
	  typeof un_objeto  // object
	  typeof otro_objeto  // object
    Object

    Un Object puede ser una coleccion de propiedades.
